## 像数 1, 2, 3 一样容易
1. `npm install fx67ll-vue-ui --save`
2. 在`main.js`中注册组件
```JavaScript
import fx67llVueUI from 'fx67ll-vue-ui'
Vue.use(fx67llVueUI)
```
3. 在vue页面中导入使用
