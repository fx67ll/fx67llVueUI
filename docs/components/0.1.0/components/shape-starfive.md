### 快速上手
```Vue
<shape-star-five shapeColor="#000000" :shapeSize="1" />
```

### 属性说明
|  参数   | 说明  |  类型  |  可选值  |  默认值  |
|  :----:  |  :----:  |  :----:  |  :----:  |  :----:  |
|  shapeColor  |  符号的颜色，十六进制颜色代码  |  String  |  -  |  #000000  |
|  shapeSize  |  符号的大小，支持数字，包括小数  |  Number  |  -  |  1  |
